Controle de CO2 eficiente e barato

O sistema de monitoramento da CO₂NTROL visa controlar os níveis de CO₂ emitidos durante o processo de fermentação do vinho tinto por meio de um sensor especializado

Monitoramento

Detecção e captura dos níveis de carbono pelo sensor.

Sistema Web

Exibição dos dados coletados pelo sensor via web.

Notificação

Em caso de anormalidade de dados, o usuário será notificado.
